# 💼 Job Salary Analysis Pipeline

An AI-powered data analysis pipeline using OpenRouter LLM.

## Pipeline Flow
Data Filter → Graphs → LLM Analysis → Final Paragraph

## Tools Used
- Tool 1: Data Filtering
- Tool 2: Insights Generator (4 graphs)  
- Tool 3: LLM Analyzer (OpenRouter)
- Final: 100 word paragraph

## Dataset
250,000 job salary records

## Model Used
meta-llama/llama-3.3-70b-instruct:free
